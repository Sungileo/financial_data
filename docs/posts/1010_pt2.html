벡터

GNP <- c(2090,2450,960)

  • GNP 라는 변수에 2090, 2450, 960 3개의 숫자 요소를 가진 벡터 생성

names(GNP) <- c("korea","japan","nepal")

  • 각 요소에 “korea”, “japan”, “nepal”의 이름 지정
GNP <- c(2090,2450,960)
names(GNP) <- c("korea","japan","nepal")

요소에 지정된 이름으로 값 출력 (인덱싱)

단일 값 출력

GNP["korea"]
korea 
 2090

벡터 사용 여러개의 값 출력

GNP[c("korea","nepal")]
korea nepal 
 2090   960

변수에 조건을 지정하기

d 라는 변수에 1~9까지의 숫자 벡터 생성

d <- c(1,2,3,4,5,6,7,8,9)
d <- c(1:9)

condi 라는 변수에 7이하 & 3이상을 만족하는 d 지정

  • TRUE, FALSE 값으로 출력
condi <- d <= 7 & d >= 3

condi
[1] FALSE FALSE  TRUE  TRUE  TRUE  TRUE  TRUE FALSE FALSE
  • 인덱싱으로 d에서의 값 출력하기
d[condi]
[1] 3 4 5 6 7

& : 두 조건을 모두 만족해야됨, (7보다 작으면서, 3보다 큰 수)

| : 두 조건중 하나만 만족하면됨, (7보다 크거나, 3보다 작거나), (엔터키 위 shift + \)

condi_2 <- d <= 3 | d >= 7
d[condi]
[1] 3 4 5 6 7

팩터

  • 팩터는 범주형 데이터를 지정하는데 사용

  • 예시)

    • bt.vct 라는 문자형 변수 벡터 지정

    • bt.fct 라는 팩터 지정

팩터 요소 확인하기

bt.vct <- c('A','B','B','O','AB','A')

bt.fct <- factor(bt.vct)

str()함수는 지정한 변수의 타입을 확인할때 사용

str(bt.vct)
 chr [1:6] "A" "B" "B" "O" "AB" "A"
str(bt.fct)
 Factor w/ 4 levels "A","AB","B","O": 1 3 3 4 2 1

levels()함수는 팩터 변수의 요소값을 확인할때 사용

levels(bt.fct)
[1] "A"  "AB" "B"  "O"

팩터의 인덱싱

bt.fct[5]
[1] AB
Levels: A AB B O
bt.fct[c(2,5)]
[1] B  AB
Levels: A AB B O

팩터,벡터를 숫자형으로 바꾸기

as.integer(bt.fct)
[1] 1 3 3 4 2 1
as.integer(bt.vct)
Warning: 강제형변환에 의해 생성된 NA 입니다
[1] NA NA NA NA NA NA

팩터에 값 추가

# 기존 6개 값을 가진 팩터에 7번째 값 추가
bt.fct[7] <- 'B'
bt.fct
[1] A  B  B  O  AB A  B 
Levels: A AB B O

요소 외 값 추가

# 팩터 요소(A, AB, B, O)에 포함되지 않은 값을 추가하면..
# Warning message = 경고(O), 에러(X)
# NA 값으로 입력됨
bt.fct[8] <- 'C'     
Warning in `[<-.factor`(`*tmp*`, 8, value = "C"): 요인의 수준(factor level)이
올바르지 않아 NA가 생성되었습니다.
bt.fct
[1] A    B    B    O    AB   A    B    <NA>
Levels: A AB B O

팩터의 요소 추가

  • levels() 함수를 써서 "C" 요소 추가

  • 8번째 값에 "C" 추가

levels(bt.fct) <- c(levels(bt.fct),"C")
bt.fct[8] <- 'C'     
bt.fct
[1] A  B  B  O  AB A  B  C 
Levels: A AB B O C

z <- matrix(1:20, nrow=4, ncol=5)

z1 <- matrix(1:20, nrow=4, ncol=5, byrow=TRUE)

행렬 붙히기 (combine)

x, y라는 변수에 4개의 값을 가진 벡터 생성

z 라는 변수에 1~20 의 값, 4행 5열의 행렬 생성

x <- 1:4
y <- 5:8
z <- matrix(1:20, nrow=4, ncol=5)

cbind() 함수는 열 기준으로 값을 붙힌다.(column bind)

rbind() 함수는 행 기준으로 값을 붙힌다.(row bind)

m1 <- cbind(x,y)
m1

m3
  [,1] [,2] [,3] [,4]

x    1    2    3    4
m4 <- cbind(z,x)

행렬의 행이름, 열이름 넣기

score <- matrix(c(90,85,69,78,85,96,49,95,90,80,70,60), nrow=4, ncol=3)
colnames(score) <- c("국어","영어","수학")
rownames(score) <- c("철수","영희","릅신","조던")
score
     국어 영어 수학

영희   85   96   80
릅신   69   49   70
조던   78   95   60
